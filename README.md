# 🌱 EcoTrack

Aplicativo mobile desenvolvido em Kotlin com foco em **sustentabilidade e conscientização ambiental**, alinhado ao conceito ESG (Environmental, Social and Governance).

---

##  Sobre o projeto

O **EcoTrack** é um aplicativo Android que incentiva hábitos sustentáveis por meio de:

- Registro de ações ecológicas diárias
- Sistema de pontuação ambiental (Eco Score)
- Informações de clima e qualidade do ar
- Conteúdos educativos sobre sustentabilidade

O objetivo é promover pequenas mudanças de comportamento que geram impacto positivo no meio ambiente.

---

##  Objetivo

O aplicativo busca incentivar práticas sustentáveis no dia a dia dos usuários através de:

- Conscientização ambiental
- Educação sobre ESG
- Gamificação de ações sustentáveis
- Informações ambientais em tempo real

---

##  Tecnologias utilizadas

- Kotlin
- Android Studio
- Jetpack Compose
- Room (banco de dados local)
- Retrofit (consumo de API)
- Coroutines
- Arquitetura MVVM
- OpenWeather API

---

##  Contexto ESG

O EcoTrack está diretamente ligado ao pilar ambiental do ESG, promovendo:

- Uso consciente de recursos naturais
- Redução de emissões de CO₂
- Educação ambiental
- Incentivo à reciclagem
- Práticas sustentáveis no dia a dia

---

##  Funcionalidades

-  Login e Cadastro de usuários  
-  Dashboard com Eco Score  
-  Informações de clima e qualidade do ar  
-  Dicas de educação ambiental (ESG)  
-  Registro de ações sustentáveis com pontuação  

---

##  Telas do aplicativo

###  Login
Acesso do usuário com e-mail e senha.

###  Cadastro
Criação de conta com nome, e-mail e senha.

###  Dashboard
Tela principal com:
- Pontuação Eco
- Informações ambientais
- Menu de navegação

###  Clima & Ar
Exibe:
- Temperatura
- Umidade
- Índice UV
- Qualidade do ar

###  Educação Ambiental
Dicas sobre sustentabilidade como:
- Transporte consciente
- Redução de plástico
- Gestão de resíduos
- Reflorestamento

###  Ações Sustentáveis
Registro de ações do usuário com sistema de pontuação:
- Transporte sustentável (+15)
- Separação de lixo (+10)
- Economia de água (+20)
- Economia de energia (+5)

---

##  API utilizada

O aplicativo consome dados da:

- OpenWeather API  
  https://api.openweathermap.org

Responsável por fornecer informações de clima e qualidade do ar em tempo real.

##  Arquitetura

O projeto segue organização em camadas:

- API (consumo de serviços externos)
- Database (Room)
- UI (Jetpack Compose)
- Screens (telas do app)
- View / Model
- Utils (funções auxiliares)
